HOW TO DOWNLOAD FROM CLAUDE CODE WEB:
================================================================

Method 1: File Browser (Left Sidebar)
--------------------------------------
1. Look at the file browser on the LEFT side of Claude Code
2. Find the ZIP files in the root directory:
   - nis2-templates-only.zip
   - nis2-templates-complete.zip
   - nis2-templates.tar.gz
3. RIGHT-CLICK on the file you want
4. Select "Download" from the context menu
5. The file will download to your browser's Downloads folder

Method 2: If Right-Click Doesn't Work
--------------------------------------
1. Click on the file name in the file browser
2. Look for a download icon or menu option at the top
3. Or use the command palette (Ctrl+Shift+P / Cmd+Shift+P)
4. Type "Download" and select the download command
